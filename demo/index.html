AKJ Carousel Demo

Dependancies

  • jQuery (tested with version 1.6.2)
  • jQuery Timers (tested with version 1.2)
  • (optional) jQuery Easing (tested with version 1.3)

Notes

  • Sizes are fixed: 1014x368 for the whole carousel, 997x354 for big images, 182x105 for thumbnails.
  • No elegant fallback in case JavaScript is diabled: the carousel will just not be displayed at all.
  • Confirmed working on: Firefox 3.6+, Google Chrome
  • Confirmed NOT working on: Internet Explorer 9+
